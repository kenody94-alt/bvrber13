
Мини-приложение для Telegram — Business дизайн
Содержимое архива:
- index.html — SPA приложение (HTML)
- styles.css — стили
- app.js — логика (vanilla JS), интеграция с Telegram.WebApp (если доступно)
- icons/ — логотип и простые SVG-иконки

Инструкция по развёртыванию:
1) Запустите статический хостинг с HTTPS (например, Vercel, Netlify, GitHub Pages with Cloudflare, любой S3+CloudFront).
2) В BotFather: /setdomain или настройте параметр "Web App" у вашего бота, указав URL на index.html (https://your-domain/path/index.html).
3) Откройте мини-приложение из чата с ботом (через кнопки или метод openWebApp) — внутри Telegram доступен объект window.Telegram.WebApp.
4) На стороне сервера можно принимать данные от клиента: в данном шаблоне TG.sendData используется для отправки JSON в бот.

Что можно доработать (рекомендации):
- Реализовать серверную часть для хранения записей (API) и уведомлений.
- Добавить аутентификацию через Telegram Bot API / серверные методы для подтверждения пользователя.
- Подключить календарь, оплату и напоминания.

Файлы готовы к редактированию в любом редакторе. Удачной интеграции!
